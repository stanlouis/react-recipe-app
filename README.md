# react_recipe-app

> A recipe search app using react to fetch recipes from the food2form API.


## Quick Start

```bash
# Install dependencies
npm install

get API key from [food2fork.com](https://www.food2fork.com/about/api)

# Serve on localhost:3000
npm start

```
